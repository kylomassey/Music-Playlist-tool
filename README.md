# Music Playlist Tool

A simple Python command-line project for modeling songs and playlists.

## Features
- Store songs with metadata
- Convert track durations to seconds
- Compute total playlist duration
- Print playlist contents
- Filter songs by genre, artist, or album

## Files
- `main.py` - entry point
- `models.py` - Song and Playlist classes
- `utils.py` - helper functions like duration formatting

## Run
```bash
python main.py
